Detailed Description

Responce object returned from the ArkaneAPI for application requests

Used by Arkane.Editor.ListApplications(ArkaneIdentity, Action<ListApplicationsResult>) to list available applications.

See https://docs-staging.arkane.network/pages/token-management.html#_list_applications for more information.
