Detailed Description

Used to create custom property objects for Tokens

Template Parameters
TMust be a serializable type for which Unity's JsonUtility can serialize

It is intended that developers will create custom property types as ScriptableObjects. These types can then be used by designers to set the property model of each indavidual Token they define.

Note that it is not nessisary to provide any body to the resulting custom property type ... this will be handled for you via the TokenProperties<T> base class.

[Serializable]
public class CharacterDataModel
{
public string name;
public string description;
public int level;
public int faction;
}
[CreateAssetMenu(menuName = "My Game/Character Properties")]
public class CharacterProperties : Properties<CharacterDataModel> { }
