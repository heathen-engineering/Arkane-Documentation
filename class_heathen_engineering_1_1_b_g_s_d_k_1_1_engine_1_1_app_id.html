Detailed Description

Responce object returned from the BGSDKAPI for application requests

Used by BGSDK.Editor.ListApplications(BGSDKIdentity, Action<ListApplicationsResult>) to list available applications.

See https://docs-staging.arkane.network/pages/token-management.html#_list_applications for more information.
