Detailed Description

Wraps the BGSDK interface for wallets incuding User, App and Whitelable wallets.

Wallet funcitonality is discribed in the https://docs.arkane.network/pages/reference.html

documentation. All functions of this class and child classes are designed to be used with Unity's StartCoroutine method. All funcitons of this class will take an Action as the final paramiter which is called when the process completes. Actions can be defined as a funciton in the calling script or can be passed as an expression.

StartCoroutine(API.Wallets.Get(Settings.user, walletId, HandleResults));

or

StartCoroutine(API.Wallets.Get(Settings.user, walletId, (resultObject) =>
{
//TODO: handle the resultObject
}));

Additional code samples can be found in the Samples provided with the package.
