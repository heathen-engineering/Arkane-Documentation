Detailed Description

Privileged features only work in restricted environments with restricted cradentials.

Privileged subsclasses such as TokenManagement.Privileged use API calls that are not available to regular end users and will always result in an error when called by a non privlaged conneciton.

Creates an instance of the indicated token

For more details please see https://docs.arkane.network/pages/token-management.html#_create_token

This requests an instance of the token be generated, the new instance will be owned by the server or developer wallet assoceated with the Arkane app and can then be transfered to the appropreate user.
