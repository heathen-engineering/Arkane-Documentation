Authenticates the indicated identity

A Unity coroutine compatable authentication method which uses an Identity to authenticate against the Arkane Networking services. For more information see https://docs.arkane.network/pages/token-management.html

Parameters
identityThe identity to authenticate
callbackA method pointer to handle the results of the authentication
Returns
  • A simple example class in the form of a MonoBehaviour which authenticates on Start

Example Output:

namespace HeathenEngineering.Arkane.Examples
{
public class ExampleBehaviour : MonoBehaviour
{
public Arkane.Settings settings;
public Arkane.Identity Identity = new Arkane.Identity();
private void Start()
{
Arkane.Settings = settings;
StartCoroutine(Arkane.Editor.Authenticate(Identity, HandleAuthenticationResult));
}
private void HandleAuthenticationResult(Arkane.DataModel.AuthenticationResult result)
{
Debug.Log("Has Error: " + result.hasError + "\nMessage:" + result.message);
}
}
}

out string  message,
bool  ignoreEmptyModel = false 
)
static

Tests the provided settings object for conflicts that could prevent processing.

Conditions tested for by this method include the following.

State Description
Empty ContractsThis is a contract with no tokens assigned to it.
Orphaned TokensThis is a token with no contract assigned to it.
Empty Contract NamesThis is a contract whoes name is empty.
Empty Contract DescriptionThis is a contract whoes description is empty.
Empty Token NameThis is a token whoes name is empty.
Empty Token DescriptionThis is a token whoes description is empty.
Duplicate Contract NameThis is a contract which shares a name with another contract.
Duplicate Contract AddressThis is a contract which shares an address with another contract.
Duplicate Token NameThis is a token which shares a name with another token.
Empty ModelThis indicates that no valid contracts and tokens have been added to the tested model.
Parameters
settingsThe settings to be tested
messageThe resulting messages
Returns
Type Description
ValidationStatus.OkayNo issues where found
ValidationStatus.Warninghas an Empty Contract or Empty Contract Description or Empty Token Description or Duplicate Contract Name or Duplicate Token Name
ValidationStatus.Errorhas an Empty Model or Empty Token Name or Empty Contract Name or Orphaned Tokens or Duplicate Contract Address
