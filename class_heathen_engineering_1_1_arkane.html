Detailed Description

The primary interface between Unity and the Arkane Networking services.

The Arkane class is defined as a partial class meaning it can be easily extended without needing to modify the original source code. Examples of this class' use can be seen throughout the Engine code provided with the package and of course in the codded documentation.

The function of the Arkane class is to serve as a wrapper around Arkane Networking's web based API. The class exposes all functions as Unity coroutine compatable methods so that they can be used in normal Unity styled code. The Arkane class is considered the 'low level API' for Heeathen's Arkane integration. It is available for use as you see fit but generally not something you would need to work with as all required funcitonality is available in higher level tools such as the ArkaneContract and ArkaneToken.
